Basic Configuration
===================

You must follow this syntax, otherwise the plugin will not load!

```
tiltify:
  token: [Insert token]
  campaigns:
    - [Insert campaign id]
    - [Insert campaign id 2]
  actions:
    [Insert action name]:
      type: [Insert action type]
      amount: [Insert amount]
      commands:
        - "[Insert Command]"
        - "[Insert Command 2]"
      messages:
        - "[Insert Message]"
        - "[Insert Message 2]"
```
