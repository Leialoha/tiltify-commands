Commands
========

These are the commands that will run on the server.

If an asterisk (*) is provide before the command, the player will send the command!

**DO NOT PROVIDE THE FIRST FORWARD SLASH!**

```
tiltify:
  ...
  actions:
    randomActionName:
      ...
      commands:
        - "[Insert Command]"
        - "[Insert Command 2]"
	  ...
```
